# Pricing Strategy & Package Descriptions (Inspired by GitHub Copilot)

เอกสารนี้ประยุกต์ใช้แนวคิดจาก **GitHub Copilot Plans** มาสู่แพลตฟอร์ม **Affiliate Link Integrity & Health Checker** โดยแบ่งกลุ่มแพ็คเกจตามสเกลความต้องการและการใช้งานจริง (Audience & Scale-based Packaging) พร้อมทั้งกำหนดรายละเอียดสิทธิ์การใช้งานเพื่อความสะดวกในการตั้งค่าใน Stripe และ Database ของระบบ

---

## 💡 แนวคิดหลักที่นำมาปรับใช้ (GitHub Copilot Concepts Applied)
1. **Audience-Focused Tiers ("For...")**: กำหนดกลุ่มเป้าหมายของแต่ละแผนการเงินให้ชัดเจนทันที เช่น "สำหรับครีเอเตอร์" หรือ "สำหรับเอเจนซี"
2. **Inheritance-Style Feature Lists ("Everything in X and...")**: ใช้การสืบทอดคุณสมบัติเพื่อให้ลูกค้าเห็นมูลค่าที่เพิ่มขึ้นชัดเจนโดยไม่ต้องอ่านข้อมูลซ้ำซ้อน
3. **Value Metric Escalation (Scale & Speed)**: ขยับขีดจำกัดตาม 2 แกนหลัก คือ **จำนวนลิงก์ (Capacity)** และ **ความถี่ในการสแกน (Frequency)**
4. **Integration Lock**: ล็อคระบบการแจ้งเตือนและการทำงานแบบครอบคลุม (API/WordPress) ไว้ในกลุ่มแผนระดับสูง เพื่อจูงใจให้ผู้ใช้สมัครแผนที่สูงขึ้น

## 📋 1. รายละเอียดสินค้าเพื่อตั้งค่าใน Stripe (Stripe Product Fields)

### 🆓 แผนที่ 1: Free (สำหรับผู้เริ่มต้นและทดสอบระบบ)
*   **Name (ชื่อแผน)**: `Free`
*   **Description (คำอธิบายในใบเสร็จ/พอร์ทัล)**:
    > สําหรับบล็อกเกอร์มือใหม่และผู้เริ่มต้นเฝ้าระวังลิงก์ (ตรวจสุขภาพลิงก์ 100 ลิงก์ สแกนอัตโนมัติวันละ 1 ครั้ง แจ้งเตือนความผิดปกติเบื้องต้นผ่าน Email ไม่ต้องผูกบัตรเครดิต)
*   **Pricing Type (ประเภทราคา)**: Recurring (จ่ายรายเดือน / ฟรี)
*   **Billing Period (รอบการเก็บเงิน)**: Monthly (รายเดือน)
*   **Amount (ยอดเงิน)**: `$0.00 USD` (ไม่ต้องตั้งค่า Stripe Plan ก็ได้ หรือใช้แบบ Default Free)
*   **Stripe Product Metadata (แนะนำใส่ใน Stripe)**:
    ```json
    {
      "plan_key": "free",
      "max_links": "100",
      "max_projects": "1",
      "scan_interval_seconds": "86400",
      "has_webhook_alerts": "false",
      "has_wp_plugin": "false",
      "has_api_access": "false"
    }
    ```

## 📊 2. ตารางเปรียบเทียบฟีเจอร์หลัก (Detailed Feature Matrix)

| ฟีเจอร์ที่รวมในแผน (Features) | **Free** ($0) | **Pro** ($10/mo) | **Pro+** ($20/mo) |
| :--- | :---: | :---: | :---: |
| **จำนวนลิงก์เฝ้าระวัง (Link Limit)** | 100 ลิงก์ | 5,000 ลิงก์ | 20,000 ลิงก์ |
| **จำนวนโปรเจกต์ (Project Limit)** | 1 โปรเจกต์ | 5 โปรเจกต์ | ไม่จำกัด (Unlimited) |
| **ความรอบสแกนอัตโนมัติ (Scan Frequency)** | ทุก 24 ชั่วโมง (วันละครั้ง) | ทุก 1 ชั่วโมง (Hourly) | ทุก 15 นาที (Real-time) |
| **แจ้งเตือนความผิดปกติ (Alert Channels)** | อีเมล (Email) เท่านั้น | อีเมล, LINE, Discord, Telegram | อีเมล, LINE, Discord, Telegram |
| **ตรวจสอบพารามิเตอร์หาย (Param Loss Check)** | ❌ ไม่มี | ซัพพอร์ต (ตรวจจับ Affiliate ID หลุด) | ซัพพอร์ต (ตรวจจับ Affiliate ID หลุด) |
| **เชื่อมต่อระบบ WordPress (WP Plugin)** | ❌ ไม่มี | ❌ ไม่มี | ซัพพอร์ต (ปลั๊กอินตรวจหลังบ้าน) |
| **สิทธิ์การใช้ API (API Client Access)** | ❌ ไม่มี | ❌ ไม่มี | ซัพพอร์ต (เชื่อม API นอกได้) |
| **จำลองภูมิภาคผู้คลิก (Geo-Targeting Checks)** | ❌ ไม่มี | ❌ ไม่มี | ซัพพอร์ต (เช็กเส้นทางแต่ละประเทศ) |
| **รายงานสีขาว (White Label Report)** | ❌ ไม่มี | ❌ ไม่มี | ซัพพอร์ต (ลบโลโก้ สำหรับส่งลูกค้า) |
| **การบริการลูกค้า (Customer Support)** | ชุมชนช่วยเหลือ (Discord/Email) | ทางอีเมลพิเศษ (Priority Email) | ซัพพอร์ตด่วน 24 ชม. (Chat / Call) |

## 🎯 3. คุณค่าการตลาดเชิงลึก (Marketing Copywriting & Value Props)

### Free Plan: **"เริ่มป้องกันรายได้รั่วไหลใน 2 นาที"**
*   **เป้าหมายหลัก**: ลดแรงต้านการทดลองใช้งาน (Frictionless Entry)
*   **ข้อความขาย (Hooks)**:
    *   *“ไม่ต้องกรอกบัตรเครดิต เริ่มต้นดูแลลิงก์ทำเงินของคุณฟรี”*
    *   *“ตรวจสอบความผิดพลาดพื้นฐาน เช่น หน้าเว็บปลายทางพัง (404/500) ก่อนที่ลูกค้าจะกดเข้าเว็บคุณ”*

### Pro Plan: **"ระบบเฝ้าระวังความปลอดภัยคอมมิชชันตลอด 24 ชั่วโมง"**
*   **เป้าหมายหลัก**: คุ้มครองผู้ทำ Affiliate ที่ต้องการสแกนบ่อยและไม่พลาดจังหวะการเปลี่ยนลิงก์แคมเปญ
*   **ข้อความขาย (Hooks)**:
    *   *“ทุกชั่วโมงที่ลิงก์แคมเปญเสีย คือเงินที่คุณทำหล่นหาย ให้เราตรวจเช็กทุก ๆ 60 นาทีและแจ้งเตือนคุณผ่านแชทแอปทันที”*
    *   *“ป้องกันปัญหา Merchant เปลี่ยนเส้นทางลิงก์และลบ Affiliate ID ของคุณออกจากการเช็กทราฟฟิก”*

### Pro+ Plan: **"ที่สุดแห่งการขยายสเกลงาน เอเจนซี และการจัดการขั้นสูง"**
*   **เป้าหมายหลัก**: ดึงดูดธุรกิจขนาดกลางหรือเอเจนซีที่ต้องการสแกนความถี่สูงมาก และใช้ระบบส่งต่อ API
*   **ข้อความขาย (Hooks)**:
    *   *“จำลองพฤติกรรมผู้ใช้จากหลายประเทศ เพื่อให้แน่ใจว่าบอทตรวจเช็กไม่หลุดแคมเปญในต่างแดน”*
    *   *“สร้างรายงานความน่าเชื่อถือในแบรนด์ของคุณเองเพื่อส่งต่อลูกค้า โดยไม่ต้องแสดงโลโก้ของระบบเรา”*

## 🛠️ 4. ข้อเสนอแนะการเชื่อมต่อทางเทคนิค (Technical Integration Note)
เมื่อเชื่อมต่อ Stripe Webhook เมื่อยอดชำระเงินสำเร็จ สามารถตั้งค่าให้ระบบอ่าน `Stripe Price Metadata` หรือส่ง API อัปเดตตาราง `tenants` และ `subscriptions` ในระบบหลังบ้านของคุณได้ทันที โดยอิงค่า `scan_frequency_seconds` ดังนี้:
*   **Free**: `86400` วินาที (24 ชั่วโมง)
*   **Pro**: `3600` วินาที (1 ชั่วโมง)
*   **Pro+**: `900` วินาที (15 นาที)
